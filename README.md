# badtaste

Play audio from [vk.com](http://vk.com/) in your terminal

![Demonstration](/media/screenshot.png?raw=true)

## Install

```
$ sudo apt-get install libasound2-dev
$ npm install -g badtaste
```

## Usage

```
$ badtaste # press '?' inside to get help
```
